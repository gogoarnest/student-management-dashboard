# Student Management Dashboard

A web-based dashboard for managing student records with authentication and database integration.
Built with HTML, CSS, JavaScript, PHP, and MySQL.

## Live Demo
https://student-dashboard.infinityfreeapp.com/login.php

## Features
- Login system
- Add and view student records
- Database (MySQL) integration
- Simple API endpoints (PHP)

## Tech Stack
HTML, CSS, JavaScript, PHP, MySQL

## Setup (Local)
1. Put the project folder in `htdocs` (XAMPP).
2. Import the SQL database into phpMyAdmin.
3. Copy `connection.example.php` to `connection.php` and update credentials.
4. Run:
   http://localhost/your-project-folder/
